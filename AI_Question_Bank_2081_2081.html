How can you relate synapse, dendrite, and axon in biological neural networks with the elements of artificial neural networks? Create a multi-layer ANN with input layer, hidden layer, and output layer. Assume necessary inputs and weights to the ANN and illustrate a single iteration of backpropagation algorithm to train the ANN.

Answer:
  • Synapse
    A synapse is the signal-transmitting link between two neurons in biological brain networks.
    Weights are used to represent the synapse in artificial neural networks. The strength and impact of the link between nodes (neurons) are determined by these weights.
  • Dendrites
    Input signals are sent from other neurons to dendrites.
    The input values given to the network are represented as dendrites in an ANN. Neurons in the network receive these inputs.
  • Axon
    Signals are sent from one neuron to the next via the axon.
    After an activation function processes the inputs, the axon in an ANN represents the neuron’s output.

 

Question 2

What is Skolem constant? How is Skolemization done during resolution? Represent the following statements into FOPL.

  • All movies are not hit.
  • Sarangi is a movie.
  • All movies which have good script are hit.
  • Sarangi has a good script but Sarangi is sentimental.
  • There is a movie which is comedy.
Answer:

Skolem constant: In First-Order Predicate Logic (FOPL), a Skolem constant is introduced during the Skolemization process when existential quantifiers (∃) are removed. A constant is used in place of an existentially quantified variable since its value is independent of any universally quantified variables.

The removal of existential quantifiers from a formula in FOPL is known as “skolemization.” A Skolem function is employed when the existential quantifier is dependent on a universal quantifier. A Skolem constant is used in place of the existential variable in the absence of a dependence.

Converting the Given Statements into FOPL

  1. All movies are not hit.
    • ∀x (Movie(x) → ¬Hit(x))
  2. Sarangi is a movie.
    • Movie(Sarangi)
  3. All movies which have a good script are hit.
    • ∀x (Movie(x)∧GoodScript(x)→Hit(x))
  4.  Sarangi has a good script but Sarangi is sentimental.
    • GoodScript(Sarangi) ∧ Sentimental(Sarangi)
  5. There is a movie which is comedy.
    • ∃x (Movie(x) ∧ Comedy(x))

Skolemization of the Statements

To remove the existential quantifier in statement 5:

  • Replace ∃x with a Skolem constant, say c:
    Movie(c) ∧ Comedy(c)

Final Skolemized Statements

  1. ∀x (Movie(x) → ¬Hit(x))
  2. Movie(Sarangi)
  3. ∀x (Movie(x) ∧ GoodScript(x) → Hit(x)) 
  4. GoodScript(Sarangi) ∧ Sentimental(Sarangi)
  5. Movie(c) ∧ Comedy(c)

Question 5

What is reinforcement learning? Configure an ANN neuron to simulate OR gate.

Answer:

Reinforcement Learning (RL) is a subfield of machine learning in which an agent learns to accomplish a goal by maximizing cumulative rewards while interacting with its environment. Through feedback in the form of incentives or penalties for its behaviors, the agent is able to gradually create the best possible strategy or policy. The foundation of RL is trial and error, with an emphasis on striking a balance between exploitation and exploration.

We set up a single perceptron ANN neuron to mimic an OR gate where:

  • Weight Vector: W = [1, 1]
  • Bias: b = -0.5
  • Input (X): Two inputs, X₁ and X₂.
  • Output: OR operation output (1 if X₁ OR X₂ is 1, else 0).

Perceptron calculates:

z = W1​X1​ + W2​X2 ​+ b

Output = Step(z) = 1 if z ≥ 0, else 0

X₁ X₁ Weighted Sum (z) Output
0 0 0×1+0×1−0.5=−0.5 0
0 1 0×1+1×1−0.5=0.5 1
1 0 1×1+0×1−0.5=0.5 1
1 1 1×1+1×1−0.5=1.5 1

Question 7

How can you represent knowledge using scripts? Create a knowlege base using script based on your own assumption.

Answer:

A script is an organized method of expressing information regarding typical circumstances, occurrences, or procedures. This kind of schema or frame explains a series of events or activities that are usually connected to a scenario. In order to simulate real-world situations, a script records the background, participants, activities, and other pertinent information. Planning and natural language processing are two fields that frequently employ scripts.

Script components:

  1. Entry conditions: Must be satisfied before events in the script can occur.
  2. Results: Conditions that will be true after events in script occur.
  3. Props: Slots representing objects involved in the events.
  4. Roles: Person involved in the events.
  5. Track: Specific variation on more general pattern in the script. Different tracks may share many components of the same script but not all.
  6. Scenes: The sequence of events that occour.

Question 11

How is minmax algorithm used in game search? Consider state space is defined by a collection of pairs like (A, B) representing paths between states A and B. Construct state space for following and use a minmax algorithm

(A, B), (A, C), (B, D), (D, E), (C, F), (C, G), (D, H), (D, I), (E, J), (F, K), (F, L), (G, M), (G, N).

The utilities for states H, I, J, K, L, M, N are 1, 3, 2, 6, 3, 4, 1 respectively.

Answer:

In game search, the Minimax method minimizes the potential loss in the worst-case situation to identify the best course of action. Using a game tree, it assesses every action that may be made and its results, alternating between maximizing the player’s gain and limiting the opponent’s gain.

A tree with nodes for states and edges for pathways can be used to represent the provided state space. A is the root node, and the terminal states with utilities are the leaf nodes. The following is the hierarchical structure with the nodes (A, B), (A, C), (B, D), (D, E), (C, F), (C, G), (D, H), (D, I), (E, J), (F, K), (F, L), (G, M), (G, N) will be:

The utilities for states H, I, J, K, L, M, N with 1, 3, 2, 6, 3, 4, 1 are:

  • For Node E :
    Value at E = J = 2

  • For Node D (Minimizer):
    Value at D = min⁡(H, I, J)= min⁡(1, 3, 2)=1
  • For Node F (Minimizer):
    Value at F = min⁡(K, L) = min⁡(6, 3) = 3
  • For Node G (Minimizer):
    Value at G = min⁡(M, N) = min⁡(4, 1) = 1

 

  • For Node B(Maximizer):
    Value at B = max⁡(D) = max⁡(1) = 1
  • For Node C (Maximizer):
    Value at C = max⁡(F, G) = max⁡(3, 1) = 3

 

 

  • For Node A (Minimizer):
    Value at A = min⁡(B, C) = min⁡(1, 3) = 1

Since, selecting option B yields a utility of 1, it is the best course of action for the player at root node A.

Question 12

Justify which type of environments resembles following agents.

  1. Mission Game with fxed 6 states having two players.
  2. Tesla Driverless Robovan where road conditions are changing.
  3. Game Result Predicting Agent where current prediction state is independent of previous state.
Answer:
  1. Mission Game with fixed 6 states having two players.
    Type of Environment: Goal-based Agent
    Justification: An agent that chooses activities to accomplish a certain objective is said to be goal-based. As the environment is finite and predictable, the agent’s choices are determined by the objective it seeks to accomplish so we can say that it is goal-based agent.
  2. Tesla Driverless Robovan where road conditions are changing.
    Type of Environment: Model-based Agent
    Justification: A model-based agent predicts how its actions will affect the environment by using an internal model of the world. In Tesla Driverless Robovan where road conditions are changing which is based on real-time data, the model-based agent uses this dynamic model to make decisions that ensure safety and optimal driving.
  3. Game Result Predicting Agent where current prediction state is independent of previous state.
    Type of Environment: Simple Reflex Agent
    Justification: A simple reflex agent just analyzes the current state while making decisions; it does not evaluate past states or actions. Since the game outcome prediction is independent of the previous state, the agent just reacts to the current input to make its forecast, as is common for a basic reflex agent.
